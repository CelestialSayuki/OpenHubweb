iOS Memory Budget Test

这是一个 iOS 小型适用应用程序，试图分配尽可能多的内存以达到崩溃，并记录内存警告和崩溃发生的时间，您可以再次运行应用程序，查看软件崩溃需要多少内存以及何时发生内存警告，这有助于了解任何 iOS 设备的内存分配预算。
GitHub 地址：https://github.com/Split82/iOSMemoryBudgetTest
IPA 安装包：MemoryTest_1.0.ipa
